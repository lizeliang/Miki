# Miki
My Wiki, powered by mistune jinja2 and gyments

# Quick Start
1. download  
```shell
git clone https://github.com/septicmk/Miki.git
```
2. write your markdown in the __post__ dir  

3. use `make` to generate you html  

4. use github-page to show your pages
